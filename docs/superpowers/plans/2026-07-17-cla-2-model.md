# CLA 2.0 and Circadian Stimulus Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the melanopic-DER proxy with the corrected CLA 2.0 model and expose real-time CS duration and visual-field conditions without changing the optimized SPD.

**Architecture:** Add an isolated spectral-data module and a pure calculation module, then adapt `app.js` to normalize the current relative SPD to corneal illuminance and display the returned values. Keep CIE S 026 melanopic metrics independent and confine DOM behavior to the existing application layer.

**Tech Stack:** Browser JavaScript, CommonJS-compatible test exports, static HTML/CSS, Node.js assertion tests, Playwright browser verification.

## Global Constraints

- Use corrected Equation 3 from the 2022 corrigendum and the official CS Calculator 2.0 constants.
- Keep CIE S 026 melanopic DER and melanopic EDI separate from CLA 2.0 and CS.
- Duration range is 0.5-3.0 hours; field factors are exactly 0.5, 1.0, and 2.0.
- Changing duration or field factor must not alter channel powers or the combined SPD.
- Preserve the one-screen desktop workspace and responsive mobile layout.
- Do not modify unrelated dirty files in the repository.

- [ ] **Step 3: Implement interpolation, illuminance normalization, corrected Equation 3, and CS**

Implement these public functions and constants:

```js
const CLA2_CONSTANTS = Object.freeze({
  normalization: 1548,
  k: 0.2616,
  blueYellow: 0.21,
  rodMel: 2.30,
  rodBlueYellow: 1.60,
  gMel: 1.00,
  gBlueYellow: 0.16,
  rodSat: 6.5215,
  halfSaturation: 355.7,
  exponent: 1.1026,
  maxCS: 0.7
});

function claToCS(cla, durationHours = 1, fieldFactor = 1) {
  const t = Math.min(3, Math.max(0.5, Number(durationHours) || 1));
  const f = [0.5, 1, 2].includes(Number(fieldFactor)) ? Number(fieldFactor) : 1;
  const stimulus = Math.max(0, Number(cla) || 0) * t * f / 355.7;
  return 0.7 * (1 - 1 / (1 + Math.pow(stimulus, 1.1026)));
}
```

The internal CLA calculation must use the corrected macular-normalized denominators, both rod terms, the blue-yellow piecewise condition, and final non-negative clipping. It must return zeros for invalid or zero-energy spectra.
